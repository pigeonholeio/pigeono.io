    Everything you need to share secrets with confidence

    PigeonHole makes sending secrets with GPG & OpenPGP Cryptography super simple.

    Chain of Trust

    PigeonHole provides the simplicity of tying a cryptographic key to an online Identity by locally generating GPG keys with secure defaults, and seamlessly leveraging your Identity Provider's user details.

    End-to-End Encryption

    Rock solid cryptography from the ProtonMail/gopenpgp library. Only the recipient's private key can decrypt the secrets you send

    Terminal Native

    No more context switching to send files!
    Built using the same powerful libraries that drive popular CLI tools like Docker, Kubernetes, Helm and CockroachDB, delivering a familiar and intuitive experience.

    Flexible Storage

    Built leveraging the S3 API protocol as the storage backend, allowing secrets of any size and quantity to be sent.

    Built for Everyone

    PigeonHole works seamlessly across macOS, Linux, Windows and Docker, ensuring you can secure and share your data regardless of your operating system.

    Transparent and Open Source

    Full transparency into how your data is handled from the the CLI tool to the backend architecture.
