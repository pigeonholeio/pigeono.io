Everything you need to share secrets with confidence

PigeonHole makes sending secrets with GPG & OpenPGP Cryptography super simple.

Chain of Trust

PigeonHole makes it simple to link a cryptographic key to your online identity. It generates GPG keys locally using secure defaults and automatically leverages your identity provider's user details.

End-to-End Encryption

Rock-solid cryptography from the ProtonMail/gopenpgp library. Only the recipient's private key can decrypt the secrets you send.

Terminal Native

Built using the same powerful libraries that drive popular CLI tools like Docker, Kubernetes, Helm and CockroachDB. PigeonHole delivers a familiar and intuitive experience.

Flexible Storage

Leveraging the industry standard S3 API protocol as the storage backend providing PigeonHole secrets of any size and quantity to be sent.

Built for Everyone

PigeonHole works seamlessly across macOS, Linux, Windows and Docker, ensuring you can secure and share your data regardless of your operating system.

Transparent and Open Source

PigeonHole provides full transparency into how secrets are handled from the CLI tool to the backend architecture to your recipient. Security should not be hidden away!
